Telegram
  1. Open Telegram, search for @BotFather and tap Start
  2. Send /newbot and follow the prompts:
    · Enter a display name (e.g. Crave Saga)
    · Enter a username — must end with bot (e.g. CraveSagaNotify_bot)
  3. BotFather will reply with your Bot Token. Copy it and paste it into the field below:
    123456789:ABCDefgh-XXXXXXXXXXXXXXXX
  4. Search for your bot's username in Telegram, open the chat and tap Start
  5. Send any message to it (e.g. hello) — this is required before the bot can message you
  6. Open the following URL in a browser, replacing <token> with the Bot Token from step 3:
    api.telegram.org/bot<token>/getUpdates
  7. In the returned JSON, find result → message → from → id — that number is your Chat ID. Paste it into the field below
  8. Enable the toggle, then click Save — your phone will receive a confirmation notification
Discord
  1. In Discord, go to the channel where you want to receive notifications and open Channel Settings
  2. Navigate to Integrations → Webhooks → New Webhook
  3. Click Copy Webhook URL and paste it into the field below
  4. Enable the toggle, then click Save — the channel will receive a confirmation message
